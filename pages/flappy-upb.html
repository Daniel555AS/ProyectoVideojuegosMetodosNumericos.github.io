¿Cómo Jugar? 🕹️

- Presiona ESPACIO para hacer que el jugador salte.
- Evita chocar contra los tubos verdes (obstáculos).
- Si chocas, presiona R para reiniciar el juego.

Método Numérico Implementado 🧮

Se utilizó la derivada numérica (aproximación de pendiente) para calcular la velocidad vertical del pájaro entre frames.

¿Para qué se Implementó el Método Numérico? 💭

Se usó para la la generación adaptativa de obstáculos.
